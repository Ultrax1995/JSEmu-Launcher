# Privacy Policy

**Last updated: July 22, 2026**

This Privacy Policy explains how the JSEmu.eu Launcher handles information when you install, use, update, or obtain support for the application.

## 2. Information stored locally

The Launcher may store the following information locally on the user's computer:

- selected game installation directory;
- launcher preferences and configuration;
- selected server information;
- names and addresses of custom servers added by the user;
- application version information;
- locally generated diagnostic logs;
- Account Key or authentication information entered by the user, where required by the selected server.

Information stored locally remains on the user's device unless it is transmitted as described in this Policy.

Users may remove locally stored information by deleting the Launcher configuration files or uninstalling the Launcher.

## 3. Information transmitted over the Internet

The Launcher requires Internet access to provide its main functions.

Depending on the functions used, the Launcher may communicate with:

- JSEmu.eu services;
- selected Just Survive game servers;
- authentication servers;
- update and file distribution servers;
- GitHub services used to distribute releases and updates.

When the Launcher connects to a network service, that service may automatically receive technical information such as:

- IP address;
- date and time of the connection;
- requested file or network endpoint;
- Launcher version;
- operating system and basic device information;
- server response and error information.

This technical information may be recorded automatically in server logs for security, troubleshooting, abuse prevention, and service maintenance.

## 4. Account Keys and authentication

When the user enters an Account Key or other authentication information, the Launcher may store it locally and transmit it to the authentication service or game server selected by the user.

Authentication information is used only to:

- verify access to a server;
- establish a game session;
- perform functions requested by the user.

Users should not share their Account Key publicly or include it in GitHub issues, screenshots, or public bug reports.

The JSEmu.eu team will never ask users to publish their Account Key.

## 5. Updates and file verification

The Launcher may automatically contact update servers to:

- check whether a newer Launcher version is available;
- download Launcher updates;
- download required game files or patches;
- verify file versions or integrity;
- retrieve the current server list;
- display project news or service information.

These connections may expose the user's IP address and standard request metadata to the server providing the requested content.

## 6. Analytics and advertising

The JSEmu.eu Launcher does not intentionally include:

- behavioural advertising;
- advertising identifiers;
- cross-application tracking;
- sale of personal information;
- profiling for marketing purposes.

The Launcher does not use personal information for direct marketing unless the user separately and explicitly agrees to receive such communications.

## 7. Diagnostic logs and bug reports

The Launcher may create diagnostic logs on the user's computer. These logs may include:

- Launcher version;
- operating system version;
- update status;
- error messages;
- file paths;
- selected server;
- connection status;
- technical details necessary to identify a problem.

Diagnostic logs are not automatically submitted unless the Launcher clearly informs the user that they will be transmitted.

Users may voluntarily submit logs, screenshots, or other technical information when requesting support or reporting a bug.

Before submitting a bug report, users should remove:

- Account Keys;
- passwords;
- authentication tokens;
- private server credentials;
- personal file paths containing their real name;
- any other information they do not wish to disclose publicly.

Public GitHub issues may be visible to anyone.

## 8. Purposes of processing

Information is processed only when necessary to:

- provide the functions requested by the user;
- authenticate the user with the selected server;
- download and verify required files;
- maintain Launcher and server security;
- prevent abuse and unauthorised access;
- diagnose technical problems;
- provide user support;
- comply with applicable legal obligations.

## 9. Legal grounds

Where the GDPR applies, information may be processed on the following legal grounds:

- performance of a service requested by the user;
- legitimate interests in maintaining the security, reliability, and availability of the Launcher and associated services;
- compliance with legal obligations;
- user consent, where consent is requested for optional data processing.

Where processing is based on consent, the user may withdraw that consent at any time.

Withdrawal of consent does not affect processing that occurred before the withdrawal.

## 10. Data sharing

Personal information is not sold.

Information may be processed by service providers necessary to operate the Launcher, including:

- hosting providers;
- update and file distribution providers;
- GitHub, where GitHub is used for source code, releases, downloads, or issue reports;
- authentication and game server operators selected by the user;
- security or infrastructure providers used by JSEmu.eu.

Information may also be disclosed where required by law or where reasonably necessary to protect users, infrastructure, or the security of the project.

Third-party services process information according to their own privacy policies.

## 11. Third-party links

The Launcher may contain links to third-party websites or services, including GitHub and Discord.

Opening such a link may cause the user's web browser to connect directly to that third party.

JSEmu.eu does not control the privacy practices of third-party websites or services.

- the user deletes it;
- the Launcher removes it during an update or reset;
- the Launcher is uninstalled and its configuration files are deleted.

Reasonable technical and organisational measures are used to protect information against:

- unauthorised access;
- accidental loss;
- misuse;
- alteration;
- disclosure.

No method of Internet transmission or electronic storage can guarantee absolute security.

Users are responsible for protecting their Account Keys, passwords, and other authentication information.

## 15. Children's privacy

The Launcher is not designed to intentionally collect personal information from children for advertising or profiling purposes.

Where a child uses a JSEmu.eu service, the child's parent or legal guardian should supervise the use of the Launcher and associated online services.

## 16. User rights

Depending on applicable law, users may have the right to:

- obtain information about the processing of their personal information;
- request access to their personal information;
- request correction of inaccurate information;
- request deletion of information;
- request restriction of processing;
- object to certain processing;
- receive portable data where applicable;
- withdraw consent where processing is based on consent;
- lodge a complaint with the relevant data protection authority.

Requests may be submitted using the contact address listed in this Policy.

Identity verification may be required before a request is completed.

## 17. Automated decision-making

The Launcher does not use personal information to make decisions producing legal or similarly significant effects through automated profiling.
